Single Shell

exit

Close the remote shell and return to Lead.

delete

Remove the current server from Lead and close its shell.

Multi Shell

Select two or more servers, open one shared terminal, and run line-based commands across all active targets at once.

Servers are assigned indexes like 0, 1, 2 based on their order when multi shell opens. Every server starts active.

A green ● means the server is an active target. A red ● means it is inactive.

docs

Show native multi-shell commands and the indexed target list inside the terminal.

all

Activate every indexed server target.

on <index>

Activate one server target without changing the others.

off <index>

Deactivate one server target.

only <index>

Deactivate every other server and keep only that target active.
